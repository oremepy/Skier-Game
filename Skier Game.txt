В создании игры использовались пакеты:

os - для нахождерия файлов в папке data

sys - для завершения программы

random - для случайной расстановки препядствий

pygame - основной пакет для создания игры


В игре мы играем за лыжника который спускается по склону объезжая
деревья и собирая флажки.

Так же есть меню с 2 кнопками и экраном окончания игры.
